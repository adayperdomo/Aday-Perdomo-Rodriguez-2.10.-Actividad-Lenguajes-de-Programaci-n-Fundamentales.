Lenguajes de Programación Fundamentales

Dennis Ritchie, creador del lenguaje C, sentó las bases para muchos de los lenguajes de programación modernos. Tres lenguajes fundamentales en la actualidad son:

  • C
  • Python
  • JavaScript

C

C es un lenguaje de programación de propósito general originalmente desarollado por Dennis Ritchie entre 1969 y 1972 en los Laboratorios Bell. Es un lenguaje imperativo de bajo nivel que proporciona acceso directo a la memoria y hardware del sistema.

Fuente: C(Lenguaje de programación)


Python

Python es un lenguaje de programación, de alto nivel y de propósito general. Su filosofía de diseño enfatiza la legibilidad del código con el uso de sangría. Sus constructos de lenguaje y su enfoque orientado a objetos tienen como objetivo ayudar a los programadores a escribir código claro y lógico.

Fuente: Python(Lenguaje de programación)


JavaScript

JavaScript es un lenguaje de programación interpretado, de alto nivel, a menudo compilado just-in-time y multiparadigma. Tiempo tipado dinámico, orientado a objetos basada en prototipos y funciones de primera clase. Es uno de los lenguajes principales de la World Wide Web, junto HTML y CSS.

Fuente: JavaScript
